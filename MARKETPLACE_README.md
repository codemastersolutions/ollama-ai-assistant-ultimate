## ⚡ Quick Start (2 Minutes Setup)

### 1. Install Ollama

```bash
# Linux/macOS
curl -fsSL https://ollama.com/install.sh | sh

# Windows: Download from ollama.com
```

### 2. Pull Required Models

```bash
ollama pull llama3.1        # Chat & general AI
ollama pull codestral:7b    # Code completion
ollama pull llava           # Image analysis
```

### 3. Install Extension

Click **[Install]** button above or search "Ollama AI Ultimate" in VSCode Extensions.

### 4. Start Using

- **Code Completion**: Automatic as you type
- **AI Chat**: `Ctrl+Shift+P` → "Chat with Ollama AI"
- **Image Analysis**: Right-click any image → "Analyze Image"
- **Security Scan**: `Ctrl+Shift+P` → "Security Scan"
